# 📚 DBMS Case Studies

🔗 Repository Link: https://github.com/rishabhchaurasia14-create/DBMS-CASE-STUDIES/tree/main

This repository contains **DBMS case studies, ER diagrams, and SQL practice files** designed to help students understand database concepts through real-world examples.

---

## 📌 About the Project

This project focuses on applying **Database Management System (DBMS)** concepts like:

* Entity-Relationship (ER) Modeling
* Relational Schema Design
* SQL Query Implementation
* Real-world Case Studies

These elements are essential in DBMS mini projects, where students design complete systems with ER diagrams and SQL operations [1])

### 🧾 SQL Files

* `DBMS Case Studies ER Projects.sql` → Main database design scripts
* `Practice 1.sql` → Basic SQL queries

### 📘 Documentation

* `README.md` → Project overview

## 🎯 Objectives

* Understand how real-world systems use databases
* Practice writing SQL queries efficiently
* Learn relationships between entities
* Prepare for **DBMS viva and exams**

---

## 🧠 Concepts Covered

* DBMS Fundamentals
* ER Diagram (Entities, Attributes, Relationships)
* Normalization
* Keys (Primary, Foreign)
* SQL Commands:

  * `SELECT`, `INSERT`, `UPDATE`, `DELETE`
  * `JOIN` (INNER, LEFT, RIGHT)
  * `GROUP BY`, `HAVING`
  * Subqueries
  * Views

## 🏗️ Case Studies

### 🛒 E-commerce Database

* Customers, Orders, Products

### 🏥 Hospital Management System

* Medical records & billing

### 🎓 University Database
